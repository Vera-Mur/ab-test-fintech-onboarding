# A/B-тест онбординга: SollmaFin
Исследование результатов А/В-теста и поиск инсайтов

---

Анализ результатов A/B-эксперимента для финтех-приложения. 
Команда обновила онбординг — добавила информацию о рисках активов. 
Задача: проверить, как это повлияло на поведение пользователей.


## Что сделано
- EDA исторических данных: воронка, сегментация по платформам и странам, ARPU
- Проверка корректности эксперимента
- Тестирование гипотез: z-test, t-test, bootstrap для перцентилей
- Рекомендация по внедрению фичи

## Результаты
- Конверсия из первого депозита во второй выросла с 20,3% до 32,6% (+61%, p < 0,001)
- ARPPU вырос на 4,3% (p = 0,0173) — статистически значимо
- Барьерная метрика не ухудшилась
- Bootstrap показал поляризацию: верхний сегмент вкладывает больше, нижний осторожнее
- Ключевая метрика (средняя сумма депозитов на пользователя) +2%, но статистически незначимо — эксперимент, вероятно, был недостаточно длительным
